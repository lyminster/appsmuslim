Cara Download File MP3 Quran

 

  1. File MP3 Qur'an harus didownload dahulu secara manual.

    Silahkan kunjungi URL sbb: http://www.everyayah.com/data/status.php atau bisa juga ke http://www.versebyversequran.com/site/download

    Kemudian klik simbol treeview (+) disebelah kiri tulisan Ayat MP3 Recitations

    Silahkan pilih link MP3 yang hendak Anda download, misalkan kita hendak download MP3 untuk Abdul Basit, maka klik link MP3 untuk Abdul Basit yang terletak disebelah kanannya.

    Setelah Anda download, simpan seluruh file MP3 pada folder MP3_Quran tempat Anda menyimpan file atau software HaditsWeb ini. Misalnya bila Anda menyimpan HaditsWeb ini pada folder sbb:

    D:\HaditsWeb\

    Maka masuklah ke subfolder sbb:

    D:\HaditsWeb\quran\MP3_Quran

    Kemudian masuklah ke folder MP3_Quran tersebut (bila tidak ada folder ini, maka Anda harus membuat folder baru dengan nama MP3_Quran)

    Simpanlah seluruh file MP3 yang telah Anda download pada folder MP3_Quran tersebut.

  2. Pastikan pada PC Anda sudah terdapat speaker dan sound card, dan sudah terinstall dengan baik drivernya. Untuk pengetesan, silahkan Anda coba jalankan salah satu file audio yang terdapat pada PC Anda, baik dalam format file mp3, maupun format lainnya, lalu coba jalankan file tersebut dengan cara di klik 2 kali, apakah terdengar suaranya melalui speaker Anda? Jika terdengar, maka Anda sudah sukses menginstall driver berikut dengan soundcard dan speakernya.

  3. Bila ActiveX pada CHM Viewer atau Web Browser Anda statusnya disable, maka harus diaktifkan dahulu ActiveX nya dengan cara klik Run ActiveX Control. Lihatlah gambar dibawah ini untuk lebih jelasnya.

 

Kesimpulan :

  1. Aplikasi Web Browser yang Anda gunakan harus mendukung penggunaan Adobe Flash Player.

  2. Penyimpanan file-file MP3 harus di folder "MP3_Quran" yang berlokasi di tempat (folder) Anda install aplikasi ini.

  3. Untuk kemudahan penggunaan, sebaiknya dibuatkan suatu shortcut di desktop, ataupun bookmark ke file index.htm (yakni halaman utama HaditsWeb ini) di browser yang Anda gunakan.
